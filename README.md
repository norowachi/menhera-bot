## About The Project

Menhera Bot is a private [Discord](https://discord.com) bot built for the Menherachan Hub & Minihub

### Built With

- [NodeJS](https://nodejs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [DiscordJS](https://discord.js.org)
- [MongoDB](https://www.mongodb.com/)
- & Other libs you can find in the package file
<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Noro95/menhera-bot-public.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your credentials in `.env` according to `.env-example`
4. Run the bot
   ```sh
   npm start
   ```
